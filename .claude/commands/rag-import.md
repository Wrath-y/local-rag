从 zip 备份文件导入向量库，替换当前全部数据（不可恢复）

输入参数：$ARGUMENTS（zip 文件路径，必填）

**第一步：确认操作**

使用 AskUserQuestion 工具向用户确认：

- 问题：`⚠️ 导入将替换当前全部向量库数据，此操作不可恢复。确认继续？`
- 选项 A：`确认导入`
- 选项 B：`取消`（推荐）

**第二步：执行导入（仅用户确认后）**

```bash
curl -s -X POST http://127.0.0.1:8765/import \
  -F "file=@$ARGUMENTS"
```

告知用户导入了多少个 chunk，以及导入是否成功。

服务未启动时提示运行 `./start.sh`。
