按来源名称删除向量库中对应的 chunk（不影响其他来源）

输入参数：$ARGUMENTS（来源名称）

**第一步：查询该来源的 chunk 数**

```bash
curl -s http://127.0.0.1:8765/sources
```

从返回结果中找到对应来源的 chunk 数。若来源不存在，告知用户并退出。

**第二步：使用 AskUserQuestion 工具展示确认选项**

向用户展示以下问题和选项，等待用户选择：

- 问题：`⚠️ 即将删除来源「{来源名称}」的全部 {chunk 数} 个 chunks，此操作不可恢复。确认继续？`
- 选项 A：`确认删除`
- 选项 B：`取消`（推荐）

**第三步：根据用户选择执行**

- 用户选择「确认删除」→ 执行删除：
  ```bash
  curl -s -X DELETE "http://127.0.0.1:8765/source?name=$ARGUMENTS"
  ```
  告知用户删除了多少个 chunk。

- 用户选择「取消」→ 告知用户已取消，不执行任何操作。

服务未启动时提示运行 `./start.sh`。
