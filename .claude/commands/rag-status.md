查看 RAG 服务状态、chunk 总数及检索命中率统计

**第一步：获取服务状态**

```bash
curl -s http://127.0.0.1:8765/health
```

**第二步：获取检索统计**

```bash
curl -s http://127.0.0.1:8765/stats
```

将两个接口的结果合并展示，包含：
- 服务状态（running / 未启动）
- 当前 chunk 总数
- rerank / verbose 开关状态
- 检索总次数、零命中次数、命中率（%）、平均每次返回 chunk 数

> 统计数据在服务重启后重置（仅内存计数）。

服务未启动时提示用户运行 `./start.sh`。
