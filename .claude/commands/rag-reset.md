清空向量库中的全部数据（不可恢复）

**第一步：查询当前 chunk 总数**

```bash
curl -s http://127.0.0.1:8765/health
```

**第二步：使用 AskUserQuestion 工具展示确认选项**

向用户展示以下问题和选项，等待用户选择：

- 问题：`⚠️ 此操作将清空全部 {chunk 总数} 个 chunks，且不可恢复。确认继续？`
- 选项 A：`确认清空`
- 选项 B：`取消`（推荐）

**第三步：根据用户选择执行**

- 用户选择「确认清空」→ 执行删除：
  ```bash
  curl -s -X DELETE http://127.0.0.1:8765/reset
  ```
  告知用户已清空。

- 用户选择「取消」→ 告知用户已取消，不执行任何操作。

服务未启动时提示运行 `./start.sh`。
